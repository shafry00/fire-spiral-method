# Fire Spiral Method — Quick Start Card

Print, fold, keep on your desk. For first-timers.

---

## STEP 1 — AMATI / Observe (5-30 min)

Rate yourself (0-10):

```
Interest:  ___   How curious?
Freedom:   ___   Tools/access/time available?
Pressure:  ___   External deadline or demand?
Gap:       ___   How deep the target? (1=surface, 10=mastery)
```

### Zone

```
WANT = (Interest + Freedom) / 2 = ___
PUSH = (Pressure + (10 - Gap)) / 2 = ___

PUSH ↑
10  │  SURVIVAL (WANT low, PUSH high)   HYBRID (mixed drivers)
 7  │ ─────────────────────────────────────────────────
    │  START (too early)                  CURIOSITY (WANT high, PUSH low)
 3  │ ─────────────────────────────────────────────────
 0  └──────────────────────────────────────────────────→ WANT
    0        5        7        10
```

## STEP 2 — PRAKTEKKAN / Practice

| Zone | Sessions | Duration | Focus |
|------|----------|----------|-------|
| Survival | 3 × 25m | 75m total | Fast, direct, exit-oriented |
| Hybrid | 3-4 × 45m | 135-180m | Selective depth |
| Curiosity | 4 × 45m | 180m total | Full exploration |

### Per Session

1. **Target 1 session** — specific, small
2. **Act** — no research, no preparation
3. **Capture gaps** — what you couldn't do

### Gap Log (per gap)

```
GAP: _____________
Thought:  _____________
Reality:  _____________
Insight:  _____________
```

### Proof Anchor (min 1 per session)

```
□ Reproduce  — Can I do it again without guidance?
□ Explain    — Can I explain it to someone who knows nothing?
□ Break      — Can I identify when NOT to use this?
```

## STEP 3 — INTEGRASIKAN / Integrate (5-30 min)

### Review all gaps → Triage

```
Go:      ___________________________  (priority, next spiral)
Review:  ___________________________  (important, shelve)
Kill:    ___________________________  (irrelevant, discard)
```

### Balance Check

```
Output (I know / I documented):  □ High  □ Balanced  □ Low
Outcome (I can / it works):     □ High  □ Balanced  □ Low

Output > Outcome → overconfidence, redo FTA
Output < Outcome → underconfidence, keep going
Balanced → accurate
```

### Done Enough?

```
□ Yes → Next (new topic) / Pick Up (dormant topic) / Deepen (same, deeper)
□ No  → What's missing? ________________
```

---

## GOLDEN RULES

- **Action first, theory later** — context comes from doing
- **Gaps = data, not failure** — every "this didn't work" is the path
- **Narrow when stuck, don't add effort** — smaller scope > more time
- **Spiral next from current bottleneck** — not from the beginning
- **Confidence before comprehension** — first goal: "I can do this"
- **Constraint Framing prevents drift** — always frame before starting

## CHEAT SHEET

| Zone | AMATI | PRAKTEKKAN | INTEGRASIKAN | Honesty Check |
|------|-------|-----------|-------------|---------------|
| Survival | Constraint Framing | Direct action, Gap Log optional | Go-only triage, fast exit | Adjusted thresholds |
| Hybrid | Hybrid Statement | Action + minimal Gap Log | Pareto + 1 Why | Standard |
| Curiosity | Full Problem Statement | Full: dismantle, experiment | Full: Insight Map, 3 Whys | Standard |

**Proof Anchor gates:** AMATI→PRAKTEKKAN (Gate 1), Gap Log→INTEGRASIKAN (Gate 2), Triage→Execute (Gate 3)
**Witness Rule:** Non-technical domains → 1 observer + 3 testers + documentation
